### A. 구글 캘린더 연동 페이지 (Detail)
> My > 구글 캘린더 연동 클릭 시 진입

#### 화면 상태 (States)
1. **연동 전 (Default)**
   - 문구: "구글 캘린더와 일정을 동기화해보세요."
   - 액션: [구글 계정으로 로그인] 버튼 노출

2. **연동 후 (Authorized)**
   - 계정 정보: 연결된 이메일 표시
   - 옵션: "일정 자동 내보내기" 스위치 (앱 → 구글 캘린더)
   - 액션: [연동 해제하기] 버튼 (빨간색)

#### 데이터 로직 (Logic)
- **로그인 성공 시:** Refresh Token 서버 저장, 화면 상태 갱신.
- **자동 내보내기 OFF:** 토큰은 유지하되, API 호출 로직만 Skip.
- **연동 해제 시:** 
    1. 서버에서 Refresh Token 삭제
    2. 화면 초기화
